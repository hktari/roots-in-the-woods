# Roots in the Woods website

### tech stack:

- gatsby
- netlify
- bootstrap
- scss
- typescript
- contentful (free plan)
- facebook graph API
- plausible

## Get started

### Developing

1. cp .env.example .env.development
2. `npm run develop`

### Deploying

1. push to `main` or `develop`
2. view https://www.rootsinthewoodsfestival.com or https://develop--rootsinthewoods.netlify.app
